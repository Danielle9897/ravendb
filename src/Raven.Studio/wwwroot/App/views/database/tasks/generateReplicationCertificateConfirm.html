Generate New Replication Certificate (public & private keys)

This option generates a new certificate for the Hub & Sink tasks communication.
The Hub task keeps only the public key.
The private key is kept only on the Sink task.

After the certificate is generated, save the private key for the Sink task definition by either one of the following:

  • Export access configuration
    All your replication access configuration (including the certificate) will be downloaded to a JSON file.
  • Download generated certificate
    Only the certificate will be downloaded as a PFX file.

Keep these files in a safe place. The private key isn't saved on the Hub task.

Please provide validity period for the generated certificate:
