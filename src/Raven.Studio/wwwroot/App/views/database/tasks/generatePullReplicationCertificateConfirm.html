Generate Replication Certificate

This option generates a new certificate for the Replication Hub & Sink tasks.
The Hub task keeps the public key.
The Sink task provides the private key when connecting to the Hub.

Save the private key for the Sink task usage by either one of the following:

  • Export Access Configuration
    All your Replication Access configuration (including the certificate)
    will be downloaded to a JSON file.
  • Download the Generated Certificate
    Only the certificate will be downloaded as a PFX file.

Keep these files in a save place. The private key isn't saved on the Hub task.

Either one of the above files can be imported when defining a Sink task.
